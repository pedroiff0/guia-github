# Issues — triagem e templates

- Use templates para bug e feature (em `.github/ISSUE_TEMPLATE`).
- Labels claros e uma rotina de triagem (ex: weekly triage) ajudam a manter backlog saudável.
- Automatize stale issues com `stale` bot para manter o repositório limpo.
- Vincule issues a PRs e a cards do Project para rastreabilidade.
