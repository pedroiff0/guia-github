# Labels recomendadas

Sugestão de labels e uso:

- `bug` — Erros e regressões
- `enhancement` — Novas funcionalidades
- `docs` — Alterações na documentação
- `good first issue` — Tarefas para novos contribuidores
- `help wanted` — Tarefas que aceitam contribuição da comunidade
- `priority:high` — Alta prioridade
- `stale` — Marcado pelo bot quando sem atividade

Defina cores e descrições no GitHub para consistência visual.
